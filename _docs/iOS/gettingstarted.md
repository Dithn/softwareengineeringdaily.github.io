We are using CocoaPods to manage dependencies.
Clone the repo and then run:
>$ sudo gem install cocoapods
$ pod install


CocoaPods requires that you open the SEDaily-IOS.xcworkspace.
>$ open SEDaily-IOS.xcworkspace
